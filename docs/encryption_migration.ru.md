# Миграция на шифрование данных

Следуйте этим шагам для миграции базы данных на использование зашифрованных полей.

1. **Обновить код**
   Получите последние изменения из репозитория.

2. **Остановить проект**
   Остановите работающее приложение и воркеры во избежание несогласованности данных.

3. **Запустить миграцию**
   Примените миграции базы данных для изменения типов полей на `EncryptedCharField`.
   ```bash
   python manage.py migrate
   ```

4. **Запустить скрипт**
   Запустите скрипт для шифрования существующих открытых данных в базе.
   ```bash
   python scripts/encrypt_all_data.py
   ```

5. **Запустить проект**
   Запустите приложение и воркеры.
